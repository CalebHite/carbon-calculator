# Carbon Calculator
To run the project in dev mode please use:<br>
`npm run dev`

To build the project for release please use:<br>
`npm run build`

To build the docker container use the provided Dockerfile:<br>
`Dockerfile`

To run using `docker-compose` use:<br>
`docker-compose up -d`
